  # Photo Guide for the Futures Church media team

  Welcome. This is where you replace photos on the live site without touching code.

  ## How to swap a photo (the only thing you need to know)

  1. Click **Photo Slots** in the left sidebar.
  2. Click the slot you want to update (e.g. *Country hero — United States*).
  3. Click the image to replace it. Upload your new file.
  4. **Important — match the filename exactly.** If the slot's current image is `usa.jpg`, name your new upload `usa.jpg` too. This keeps the live site working.
  5. Click **Save** in the top right.
  6. Your change is live in about 90 seconds.

  ## Where photos live

  - **Pastor cards** → `/photos/pastors/`
  - **Country heroes** (campuses page) → `/photos/countries/`
  - **Campus photos** (individual campus cards) → `/photos/campuses/`
  - **Homepage hero** → `/photos/hero/`
  - **Gallery & mosaic** → `/photos/gallery/`, `/photos/mosaic/`

  ## File naming rules

  - Lowercase only — `alpharetta.jpg`, never `Alpharetta.JPG`
  - No spaces — use hyphens — `ashley-jane.jpg`, never `Ashley Jane.jpg`
  - Always `.jpg` for photos (use `.png` only for logos/graphics)
  - Replace, don't add — the goal is to overwrite the existing file with the same name

  ## Need a brand-new slot (a new pastor, a new campus)?

  That requires a small code change — message Ashley and we'll add the slot in 5 minutes.

  ## Brand directive

  This site competes with the best sites in the world. Photos should feel **cinematic, editorial, modern** — not like a typical church. When in doubt, choose: cleaner, simpler, more space.
